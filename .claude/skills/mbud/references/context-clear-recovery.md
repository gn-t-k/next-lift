# コンテキストクリア復帰手順

コンテキストクリア後にスキルの作業を再開する場合、以下の手順に従う。

## 復帰手順

1. **出力ファイルを読み込む**: ユーザーに出力ファイルのパスを確認し、全文を読み込む
2. **ステータスを確認する**: 「ステータス」セクションの「現在のフェーズ」と「次のアクション」を把握する
3. **設計判断ログを確認する**: これまでにどのような判断がなされたかを理解する
4. **パーキングロットを確認する**: 未対応の項目を把握し、該当フェーズで対応する
5. **ユーザーに状況確認する**: 以下を確認してから作業を再開する
   - 現在のフェーズが正しいか
   - 追加の指示や方針変更がないか
   - 中断前に議論中だった事項があれば確認
6. **作業を再開する**: 確認後、該当フェーズのサブエージェントを起動する

## 注意事項

- 出力ファイルに記録されていない口頭での合意事項がないか、必ずユーザーに確認する
- フェーズの途中で中断していた場合は、フェーズの最初からやり直す（出力ファイルの該当セクションをサブエージェントが読み直す）
